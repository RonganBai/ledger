# Supabase Edge Functions

Before deploying, set these secrets in Supabase:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `FEEDBACK_SMTP_HOST`
- `FEEDBACK_SMTP_PORT`
- `FEEDBACK_SMTP_USERNAME`
- `FEEDBACK_SMTP_PASSWORD`
- `FEEDBACK_MAIL_FROM`
- `FEEDBACK_MAIL_TO`

Recommended QQ Mail settings:

- `FEEDBACK_SMTP_HOST=smtp.qq.com`
- `FEEDBACK_SMTP_PORT=465`
- `FEEDBACK_SMTP_USERNAME=<your qq mailbox>`
- `FEEDBACK_MAIL_FROM=<same as username>`
- `FEEDBACK_MAIL_TO=ronganbai01@gmail.com`
