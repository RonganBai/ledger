# Ledger App 使用教程

本教程覆盖当前版本已经上线的主要功能，包括日常记账、云同步、用户反馈和管理员后台。

## 1. 开始使用

### 登录或注册

1. 打开应用。
2. 如果你已经有账号，选择 `登录`。
3. 如果你需要新账号，选择 `注册`。
4. 输入邮箱和密码。
5. 如果系统启用了邮箱验证，请先完成邮箱验证再登录。

### 游客模式

你也可以选择 `游客使用`。

游客模式适合临时本地使用，但有这些限制：

- 不能使用 Supabase 云同步
- 不能管理账号邮箱或密码
- 某些和账号相关的功能不可用

## 2. 语言和主题

应用支持中文和英文。

- 在登录页右上角可以切换语言。
- 登录后可以在设置页切换语言和主题。
- 还可以调整主题风格、背景图片和背景雾化强度。

## 3. 记账基础

### 账户

账户是账单的归属容器。

常见示例：

- 现金
- 银行卡
- 支付宝
- 微信支付

目前支持的账户操作：

- 创建账户
- 重命名账户
- 调整排序
- 启用或停用账户
- 切换当前账户

现在每个已登录用户的本地账户都是隔离的，不会再出现一个用户看到另一个用户本地账户的问题。

### 账单

你可以在记账首页新增和管理账单。

支持的账单方向：

- `income`
- `expense`
- `pending`

常见账单字段包括：

- 金额
- 日期时间
- 商户
- 备注
- 分类

## 4. 报表与分析

报表模块可以帮助你查看财务变化情况。

目前支持：

- 趋势图
- 分类占比
- 历史汇总
- 按时间维度聚合分析

你可以通过这个页面查看支出结构、收入变化和长期趋势。

## 5. 外部账单导入

应用支持从外部平台导入账单。

目前支持的来源：

- 微信支付
- 支付宝
- PNC

通用流程：

1. 先从原平台导出账单文件。
2. 打开应用中的导入页面。
3. 选择对应的平台类型。
4. 选中导出的文件。
5. 预览后导入记录。

## 6. 定期交易

你可以创建重复发生的收入或支出。

典型场景：

- 工资
- 房租
- 订阅扣费
- 固定月转账

定期交易功能可以减少重复手工录入。

## 7. 云同步

登录用户可以使用 Supabase 云同步。

主要行为包括：

- 登录后下载云端账单
- 将本地改动上传到云端
- 保持本地和云端数据一致

如果当前登录用户在云端没有对应账户，应用会为当前用户建立正确的本地账户环境，而不是复用其他账号的本地账户。

## 8. 账号资料

用户资料页用于管理账号身份信息。

当前规则：

- 名称默认使用注册邮箱
- 你后续可以改成自定义名称
- 管理员后台和反馈记录会使用这个公开名称来区分用户

## 9. 用户反馈

设置页中包含反馈入口。

### 如何发送反馈

1. 打开设置页。
2. 点击右上角反馈图标。
3. 在大文本框中输入反馈内容。
4. 点击 `发送反馈`。

### 反馈规则

- 每个用户每天最多可发送 5 次反馈
- 刷新时间按 `UTC+8`
- 反馈通过服务端邮件功能发送
- 管理员后台可以查看、筛选并标记反馈为已处理

如果你没有设置自定义名称，系统会使用脱敏后的注册邮箱来区分不同用户。

## 10. 被封禁账户

管理员可以禁用用户账号。

如果账号已被封禁：

- 你不能继续使用受保护功能
- 系统在检测到访问时会强制退出登录
- 登录页会弹出“账户已被封禁”的提示框

## 11. 管理员登录流程

管理员和普通用户共用同一套登录系统。

管理员流程如下：

1. 先用普通账号方式登录。
2. 系统检查该用户是否在管理员表中。
3. 如果是管理员，会进入 4 位数管理员 PIN 二次确认页。
4. PIN 验证成功后，进入管理员双入口页。

双入口页包括：

- 应用入口
- 管理员后台入口

## 12. 管理员后台

管理员后台目前分为多个区域。

### Dashboard 总览

总览页会显示：

- 未处理反馈数量
- 疑似高频反馈用户数量
- 请求趋势图
- 总用户数
- 管理员数量
- 被禁用用户数量

### 用户权限页面

管理员可以：

- 按邮箱、昵称、脱敏邮箱搜索
- 按角色和反馈状态筛选
- 将普通用户升级为管理员
- 将其他管理员降级为普通用户

安全保护：

- 当前管理员不能降级自己
- 最后一个有效管理员不能被移除

### 用户封禁页面

管理员可以：

- 禁用用户
- 恢复被禁用用户
- 查看禁用原因

安全保护：

- 当前管理员不能禁用自己
- 最后一个有效管理员不能被禁用

### 用户反馈页面

管理员可以：

- 查看最近反馈
- 按状态筛选
- 按用户或内容搜索
- 将反馈标记为已处理
- 打开对应用户详情页

### 用户详情页

用户详情页会显示：

- 注册邮箱
- 名称
- 用户 ID
- 注册时间
- 角色状态
- 封禁状态
- 反馈统计
- 最近反馈
- 审计日志

## 13. 审计日志

重要的管理员操作都会被记录。

当前记录的操作包括：

- 授予管理员
- 移除管理员
- 禁用用户
- 恢复用户
- 处理反馈

## 14. 版本说明

本教程对应的是 `2.x` 版本线，也就是管理员系统和反馈系统的大版本更新之后的当前功能集合。
